The Sunrise Press

This is a news website built on the Ruby on Rails framework. It allows visitors to view both inhouse and sourced news articles.

On this project we learned all about API's, login and logout, subscriptions, Stripe, so on and so forth. This was a huge feet for us considering we'd only been doing this for three months.
